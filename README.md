# ProductAttachment Module
[![Latest Stable Version](https://poser.pugx.org/spryker/product-attachment/v/stable.svg)](https://packagist.org/packages/spryker/product-attachment)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.3-8892BF.svg)](https://php.net/)

ProductAttachment module is for managing product attachments linked to product abstracts.

## Installation

```
composer require spryker/product-attachment
```
